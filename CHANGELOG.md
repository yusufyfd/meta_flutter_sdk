## 0.4.0

- Add Flutter UIScene lifecycle support for iOS URL callbacks.
- Add structured, nullable error details to cancelled and failed login results.
- Treat iOS Limited Login authentication tokens as successful login results.
- Require Flutter 3.38.0 or newer for UIScene lifecycle APIs.

## 0.3.0

- Add typed `MetaLoginPermission` values.
- Add `isLogged`, `isExpired`, and `isSuccess` convenience helpers.

## 0.2.0

- Add constants for standard Meta App Event names.
- Add support for reading the anonymous App Events ID.
- Add support for setting and clearing the App Events user ID.

## 0.1.0

- Add explicit native Facebook SDK initialization for Android and iOS.
- Add Facebook Login and iOS Limited Login.
- Add App Events, flush behavior, privacy flags, and data processing options.
- Add current access token and raw Graph API requests.
- Pin Android SDK 18.2.3 and iOS SDK 18.0.3.
